# Projeto: Cartão de Visita em ASCII Art

`Objetivo:` Utilizar exclusivamente a função print() para criar uma representação visual de um cartão de visita ou uma figura simples (ASCII art) no terminal.

`Requisitos:`
1.	Use múltiplas instruções print() para desenhar a sua criação.
2.	Cada linha do desenho deve corresponder a uma chamada da função print().
3.	O projeto deve ser executável e exibir o resultado na consola sem erros.
